Kebijakan Privasi — Terakhir diperbarui: April 2026

📦 1. Penyimpanan Data

Semua data disimpan secara lokal di perangkat Anda menggunakan database SQLite. Kami tidak mengumpulkan, mengirim, atau menyimpan data pribadi apa pun di server eksternal. Data Anda tetap berada di perangkat Anda setiap saat.

🔒 2. Izin Akses

  • Penyimpanan: Digunakan untuk menyimpan dan mengekspor data arisan.

Aplikasi ini berfungsi 100% offline dan tidak memerlukan koneksi internet untuk fitur utamanya.

📈 3. Iklan

Kami menggunakan Google AdMob untuk menampilkan iklan. AdMob mungkin mengumpulkan pengenal perangkat untuk personalisasi iklan. Anda dapat menonaktifkan iklan yang dipersonalisasi di pengaturan perangkat Anda (Pengaturan > Google > Iklan > Nonaktifkan Personalisasi Iklan).

🗑 4. Penghapusan Data

Anda dapat menghapus semua data kapan saja di dalam aplikasi. Menghapus instalasi aplikasi akan menghapus semua data yang tersimpan secara permanen dari perangkat Anda.

👶 5. Privasi Anak-anak

Aplikasi ini tidak secara sengaja mengumpulkan informasi pribadi dari anak-anak di bawah 13 tahun. Semua data disimpan secara lokal dan tidak dapat diakses oleh kami atau pihak ketiga mana pun.
