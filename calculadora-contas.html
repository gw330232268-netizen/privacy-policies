Politica de Privacidade — Ultima atualizacao: Abril 2026

📦 1. Armazenamento de Dados

Todos os dados sao armazenados localmente no seu dispositivo usando um banco de dados SQLite. Nao coletamos, transmitimos ou armazenamos nenhum dado pessoal em servidores externos. Seus dados permanecem no seu dispositivo o tempo todo.

🔒 2. Permissoes

  • Armazenamento: Usado para salvar e compartilhar resultados de calculos.

Este aplicativo funciona 100% offline e nao requer conexao com a internet para suas funcionalidades principais.

📈 3. Publicidade

Usamos o Google AdMob para exibir anuncios. O AdMob pode coletar identificadores do dispositivo para personalizacao de anuncios. Voce pode desativar anuncios personalizados nas configuracoes do seu dispositivo (Configuracoes > Google > Anuncios > Desativar Personalizacao de Anuncios).

🗑 4. Exclusao de Dados

Voce pode excluir todos os dados a qualquer momento dentro do aplicativo. Desinstalar o aplicativo removera permanentemente todos os dados armazenados do seu dispositivo.

👶 5. Privacidade de Criancas

Este aplicativo nao coleta intencionalmente informacoes pessoais de criancas menores de 13 anos. Todos os dados sao armazenados localmente e nao sao acessiveis por nos ou por terceiros.
